## Célula Imobiliária

Sistemas para imobiliárias e corretores


## Características

 - Aluguel (Contratos, Recibos, Relatórios, Chaves, Fotos, Exportação para internet, Anúncios em Jornais e muito mais...);
 - Carteira de Vendas (Contratos, Financeiro, Comissões, Loteamentos, Plantas e muito mais...);
 - Gera DIMOB;
 - Gera Boletos para os principais bancos do País (Também emite arquivos de Remessa & Retorno);
 - Também integrado com CobreBem;
 - Sistema Cliente/Servidor Windows (XP, Vista, 7, 8 e 10) (32/64 bits);
 - Base de dados FirebirdSQL 2.1 (Windows / Linux);
 - Permissões por usuários 100% personalisáveis; 
 - Integração com os maiores portais do Brasil;
 - Interface online (JSON) REST para acesso em outros aplicativos;
 - Atualizações automáticas online
 - Possibilidade de acesso remoto ao sistema (Corretores Externos com conexão 3G por exemplo)
